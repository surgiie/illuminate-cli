# stringable

illuminate cli component for [laravel stringable](https://laravel.com/docs/12.x/strings#fluent-strings-method-list)

## Use

You can use the `illuminate support:stringable` command to create a stringable instance and apply various methods to it as supported by the `Illuminate\Support\Stringable` class:

```bash
illuminate support:stringable foo --method=upper

# returns "FOO"
```

## Syntax

Refer to [syntax documentation](/docs/syntax.md) for more details on the syntax and how to call methods, pass arguments, cast input and more.
